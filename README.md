# C++ Data Structures, Algorithms, and OOP Foundations

This repository contains core implementations of fundamental data structures and algorithms developed in C++. It serves as a foundational baseline for algorithmic efficiency and Object-Oriented Programming (OOP) principles.

## Repository Architecture

* **`OOP/`**: Demonstrates real-world modeling using Multiple Inheritance, Polymorphism (pure virtual overrides), and strict Encapsulation (e.g., Bank Account systems, Library Management).
* **`Data_Structures/`**: Contains custom memory-efficient data structures, including Queue generation via dual Stacks, in-place Stack manipulation, and Palindrome validation algorithms.
* **`Algorithms/`**: Focuses on search and sort optimizations, featuring both iterative and recursive Binary Search implementations, alongside string manipulation and sorting logic.

**Tech Stack:** C++
